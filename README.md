# Bakery Packaging Website

This is a frontend-only website for a bakery packaging manufacturer.

## Pages

*   Home
*   Products
*   Services
*   Client Reviews
*   Contact

## Design

The design is inspired by Hobnob.pk, with a premium and classy aesthetic. The color palette is:

*   Maroon: `#7B2D26`
*   Creamy / Off-white: `#FFF7EE`
*   Navy blue: `#032B44`
*   Yellow accents: `#FFD24C`

## How to use

1.  Clone the repository.
2.  Open the `index.html` file in your browser.
3.  To replace the placeholder images, go to the `images` folder and replace the files with your own images. Make sure the file names match the ones in the HTML files.

## Dependencies

This is a static website built with HTML, CSS, and JavaScript. There are no dependencies to install.
